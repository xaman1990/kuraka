2. `sie_v2/.claude/rules/02-clean-code.md` - Clean code, naming, early return
3. `sie_v2/.claude/rules/03-file-organization.md` - File size limits (components < 400 lines)

Do NOT read backend rules (01, 04, 05, 07, 08, 10-16) — they are not relevant for frontend work.
